## AQUARIUS

set of python helper functions that read in Aqurius csv data files into a python pandas
dataframe



See examples/aquarius_example.py for usage

Requirements:
--------------

* pandas

Installation:
------------

::

	$ git clone https://github.com/OneGneissGuy/aquarius.git
	$ cd aquarius
	$ python setup.py install
